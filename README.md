## Track mania leaderboard app for facts
This app is build with Meteor.js and react.js.

### Features
- Show top 10 players
- Show Latest 5 entries
- Add players
- Edit players
- Search for players

###Structure
- Client
    - components
        - componentx
            - jsx file
            - scss file
- Imports
    - model
    - routes
- Public
    - images
    - lib (vendor libraries)
- Server
